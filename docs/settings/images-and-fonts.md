# 图片与字体

逐控件说明见[图片设置](/settings/image)和[字体设置](/settings/font)。本页用于理解两类设置如何共同影响阅读密度。

## 图片比例

帖子图片比例允许 50%–200%，可以输入自定义值，也可以使用常用预设。浮窗、全屏和移动分别保存；开启共享后使用同一比例。

图片比例不影响：

- 灯箱中的原图尺寸；
- 下载文件；
- LINUX DO 原始帖子内容；
- 宿主页面图片。

## 灯箱默认行为

| 设置 | 默认 | 说明 |
| --- | --- | --- |
| 默认展开图片评论 | 开 | 打开灯箱后直接显示右侧评论面板 |
| 默认展开图片描述 | 开 | 显示描述区域，并记住拖动后的高度 |
| 默认使用原图 | 关 | 关闭时先加载预览，需要时手动取原图 |

描述高度最小 56 px，默认 120 px。评论面板宽度默认 25%，允许 18%–50%。

## 内置字体渲染

“内置字体渲染”改善阅读器的字体描边与平滑表现。默认同时扩展到 LINUX DO 宿主页面。

若检测到外部字体渲染脚本，设置中心会显示环境状态。为了避免多套方案叠加，出现字重异常、发虚或过度描边时，先只保留一套字体渲染。

## 宿主主题列表尺寸

可分别调整 50%–200%：

- 标题，默认 105%；
- 头像，默认 70%；
- 右下角信息，默认 105%；
- Label Card，默认 105%。

这些设置只作用于阅读器增强后的宿主主题列表，不改变阅读器正文。

## 字体作用域

阅读器内有三个独立作用域：

| 作用域 | 控制内容 |
| --- | --- |
| 界面 | 标题栏、按钮、设置和辅助信息 |
| 正文 | 帖子、引用和楼中楼正文 |
| 回复 | 回复编辑器及其输入区域 |

每个作用域都可设置：

- 字体：跟随站点、系统字体、中文无衬线、衬线、等宽或自定义本机字体；
- 字重：300、400、500、600；
- 颜色：跟随主题或自定义；
- 字号：50%–250%。

回复字号默认比界面和正文更紧凑。自定义字体只引用本机已安装字体，不会上传字体文件。

## 跨形态共享

字体、字重、颜色和字号都能单独共享。常见配置：

- 字体家族共享，字号按形态独立；
- 正文颜色共享，界面颜色按形态独立；
- 移动端使用更小字号，浮窗/全屏保持一致。
