# 布局设置

路径：**阅读器标题栏 → 设置 → 布局设置**。

![布局设置页中的三种形态与五区比例](/screenshots/guide-05-layout-settings.png)

## 五个区域

布局从左到右始终由五区组成：

| 区域 | 用途 | 下限 |
| --- | --- | ---: |
| 左留白 | 正文左侧呼吸空间 | 0% |
| 主帖 | 正文、楼中楼和主题底部 | 40% |
| 中间间距 | 主帖与时间轴之间的距离 | 0% |
| 时间轴 | 楼层位置、日期和进度 | 6% |
| 右留白 | 时间轴外侧空间 | 0% |

五项合计必须是 100%。拖动某一项时，阅读器自动从其他可调区域补偿，不会产生超出或不足 100% 的布局。

## 默认比例

| 形态 | 左留白 | 主帖 | 间距 | 时间轴 | 右留白 |
| --- | ---: | ---: | ---: | ---: | ---: |
| 浮窗 | 10% | 72% | 3% | 10% | 5% |
| 全屏 | 20% | 58% | 5% | 9% | 8% |
| 移动 | 10% | 68% | 3% | 12% | 7% |

## 调整步骤

1. 选择“浮窗 / 全屏 / 移动”。
2. 先调整主帖宽度。
3. 再调整时间轴，确保楼层号和日期完整。
4. 用中间间距控制正文与时间轴的分隔。
5. 最后分配左右留白。
6. 点击“确认应用”。

## 链状共享

每一行右侧的链按钮只共享该区域，不是共享整套布局。

例如，开启“时间轴”共享后，三种形态使用同一时间轴比例；左留白、主帖等仍可各自保存。共享项目变化时，当前形态的其余区域继续自动平衡。

## 判断是否合适

- 正文频繁换行：增加主帖比例。
- 楼层日期截断：增加时间轴比例。
- 阅读焦点过散：适当增加左右留白。
- 楼中楼关系线拥挤：增加主帖，减少无必要留白。
- 移动端横向空间不足：优先保留主帖和时间轴。

“恢复默认”只恢复当前布局草稿；确认应用前可以继续调整。
