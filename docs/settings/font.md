# 字体设置

路径：**阅读器标题栏 → 设置 → 字体设置**。

![字体设置页中的渲染、宿主文字和分作用域字体控件](/screenshots/guide-04-font-settings.png)

## 字体渲染

| 开关 | 作用范围 |
| --- | --- |
| 内置字体渲染 | 阅读器界面和正文 |
| 扩展到 LINUX DO 宿主页面 | 主题列表、原生主题页等宿主区域 |

若页面还安装了其他字体渲染脚本，面板会提示环境状态。出现字重发黑、文字发虚或描边重复时，先只保留一套渲染，再刷新页面比较。

## 宿主主题列表文字

宿主尺寸不会改变阅读器正文：

| 项目 | 默认 | 范围 |
| --- | ---: | ---: |
| 标题 | 105% | 50%–200% |
| 头像 | 70% | 50%–200% |
| 右下角信息 | 105% | 50%–200% |
| Label Card | 105% | 50%–200% |

修改后立即观察左侧原站列表，确认没有拥挤、截断或信息重叠。

## 三个字体作用域

| 作用域 | 典型内容 |
| --- | --- |
| 界面 | 标题栏、按钮、设置、时间轴和辅助信息 |
| 正文 | 帖子、引用、楼中楼、Onebox 和提示块 |
| 回复 | 回复编辑器及输入区域 |

每个作用域都可以设置字体、字重、颜色和字号。调整正文不会强制改变界面；回复编辑器也可以保持更紧凑的字号。

## 字体选项

- 跟随站点；
- 当前系统字体；
- 中文无衬线；
- 衬线；
- 等宽；
- 自定义本机字体。

自定义字体只引用本机已安装的字体名称，不会上传字体文件，也不会让其他设备自动获得该字体。

## 配置步骤

1. 选择浮窗、全屏或移动。
2. 选择界面、正文或回复作用域。
3. 依次设置字体、字重、颜色、字号。
4. 需要跨形态一致的字段，单独开启链状共享按钮。
5. 点击“确认应用”。

每个字段独立共享。例如，可以共享字体家族和颜色，但让移动端字号更小。

## 调整建议

- 先定字体和字重，再调字号，最后调颜色。
- 正文字重优先使用 400 或 500；过高字重会压缩长帖阅读层次。
- 自定义颜色应在明、暗两种主题下分别检查对比度。
- 只出现宿主页面异常时，关闭“扩展到 LINUX DO 宿主页面”，无需关闭阅读器内渲染。
