# 更新记录

## 0.1.3 — 阅读队列与主题配色

核验日期：2026-07-23。

- 新增多主题阅读队列、后台分批预加载、阅读位置恢复、固定与清理操作。
- 外观颜色自动生成明亮/暗色适应色，并支持单独关闭结构配色而保留原配置。
- 完善预加载楼层的已读同步、进入动画，以及虚拟流向上加载时的锚点稳定性。
- 收窄首楼分隔线和特殊帖子样式，避免无关内容被误改。

## 0.1.2 — 正式手册基线

核验日期：2026-07-23。

本版首次建立可自动验证的正式用户手册和功能目录。目录覆盖当前源码中的用户可见能力：

- 五种阅读形态、多主题队列、虚拟化连续阅读与楼中楼；
- 时间轴、历史前后切换、只看楼主、父回复和引用上下文；
- 原图灯箱、评论、下载、音视频、HLS、公式、投票和代码预览；
- 回复、点赞、回应、Boost、收藏、通知级别和权限操作；
- 消息、历史、收藏/回应、用户卡与关系操作；
- 13 个设置面板、分形态配置和按字段共享；
- 缓存、资源监控、请求数据、429 协调和 Cloudflare 恢复；
- 安装、隐私、兼容性和故障排查。

::: info
这里的“基线”表示已用当前源码和运行界面核对，不代表每项功能都在 0.1.2 才首次实现。缺少可靠历史证据时，不反推早期引入版本。
:::

## 0.1.0 — 早期公开介绍

仓库曾以 `0.1.0` 维护项目介绍素材，但没有可验证的逐功能更新记录。本手册不据此推断旧版功能清单。

## 维护规则

后续每次用户可见变化必须同步更新：

1. 对应功能项的 `version`、`last_verified`、`source_anchor` 和文档映射；
2. 受影响页面 frontmatter；
3. 本更新记录；
4. 必要的脱敏截图；
5. `npm run docs:verify` 结果。
